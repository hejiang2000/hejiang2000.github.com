登机桥软件设计

登机桥状态查询接口

1. 查询登机桥当前状态

请求:

GET /api/status?api_key=API_KEY HTTP/1.1

获取失败, 返回:

HTTP/1.1 401 Unauthorized

获取成功, 返回:

HTTP/1.1 200 OK
Content-Type: application/json; charset=utf-8 

{
    bridges: [{
        bridge: "1号桥",
        status: "开启"
    },{
        bridge: "2号桥",
        status: "关闭"
    }]
}
