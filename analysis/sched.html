登机桥软件设计

调度子系统功能

  1. 显示内容

    1. 航班机位信息表，显示航班接机计划和变更
    2. 候机楼俯视图，显示登机桥状态

  2. 手持机确认操作调度

    1. 任意待确认状态发出后，操作员底色从白色改为黄色
    2. 收到操作员确认信息，将操作员底色从白色改为绿色
    3. 收到操作员拒绝信息，将操作员底色从白色改为红色，同时报警
    4. 调度室工作人员手动操作，更换操作员
    5. 发出调度5分钟无响应，将操作员底色从白色改为红色，同时报警
    6. 操作员无法响应，调度室可以主动进行确认

  3. 手持机提供信息调度???
    例如，操作人员到达这个状态需要手持机提供，但如果手持机人员一直未提供，怎么处理

  4. 显示内容变更

    1. 十五分钟后到达，航班机位信息表中航班底色白色改成绿色
    2. 接机计划变更后，更新航班机位信息表
    3. 根据登机桥状态，更新航班机位信息表
    4. 根据登机桥状态，更新候机楼俯视图
