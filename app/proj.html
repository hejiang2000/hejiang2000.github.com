PROJECTS

获取全部项目信息

请求

GET /api/projects?api_key=MASTER_KEY

返回

HTTP/1.1 200 OK
Content-Type: application/json; charset=utf-8 

[
  {
    "id":"<PROJECT_ID>",
    "name":"我的健身房",
    "read_key":"<READ_KEY>",
    "write_key":"<WRITE_KEY>",
    "collections_url":"/api/projects/<PROJECT_ID>/collections",
    "collections":[
      {
        "name": "trainers",
        "url": "/api/projects/<PROJECT_ID>/collections/trainers"
      },
      {
        "name": "students",
        "url": "/api/projects/<PROJECT_ID>/collections/students"
      },
      {
        "name": "classes",
        "url": "/api/projects/<PROJECT_ID>/collections/classes"
      }
    ],
    "url":"/api/projects/<PROJECT_ID>",
    "queries_url":"/api/projects/<PROJECT_ID>/queries"
  },
  { ... },
  { ... }
]

创建一个新项目

请求

POST /api/projects?api_key=MASTER_KEY
Content-Type: application/json; charset=utf-8 

{
    "name":"我的健身房",
    "application": 1
}

返回

HTTP/1.1 200 OK
Content-Type: application/json; charset=utf-8 

{
    "id":"<PROJECT_ID>",
    "name":"我的健身房",
    "read_key":"<READ_KEY>",
    "write_key":"<WRITE_KEY>",
    "collections_url":"/api/projects/<PROJECT_ID>/collections",
    "collections":[
      {
        "name": "trainers",
        "url": "/api/projects/<PROJECT_ID>/collections/trainers"
      },
      {
        "name": "students",
        "url": "/api/projects/<PROJECT_ID>/collections/students"
      },
      {
        "name": "classes",
        "url": "/api/projects/<PROJECT_ID>/collections/classes"
      }
    ],
    "url":"/api/projects/<PROJECT_ID>",
    "queries_url":"/api/projects/<PROJECT_ID>/queries"
}

更新项目

请求

PUT /api/projects/PROJECT_ID?api_key=MASTER_KEY
Content-Type: application/json; charset=utf-8 

{
    "name":"我的健身房 2"
}

返回

HTTP/1.1 200 OK

删除项目

请求

DELETE /api/projects/PROJECT_ID?api_key=MASTER_KEY

返回

HTTP/1.1 200 OK
