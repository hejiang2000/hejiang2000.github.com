COLLECTIONS

读取全部集合信息

请求

GET /api/projects/PROJECT_ID/collections?api_key=READ_KEY

返回

HTTP/1.1 200 OK
Content-Type: application/json; charset=utf-8

[
  {
    "name": "trainers",
    "properties": {
      "name": "number",
      "item.on_sale": "boolean",
      "item.price": "number",
      "quantity": "number",
      "screen.name": "string",
      "user.id": "number",
      "user.level": "number",
      "user.referring_source": "string"
    },
    "url": "/api/projects/<PROJECT_ID>/collections/trainers"
  },
  {
    "name": "students",
    "properties": {
      "from_level": "number",
      "level": "number",
      "screen.name": "string",
      "to_level": "number",
      "user.id": "number",
      "user.level": "number",
      "user.prior_balance": "number",
      "user.referring_source": "string"
    },
    "url": "/api/projects/<PROJECT_ID>/collections/students"
  },
  {
    "name": "classes",
    "properties": {
      "user.email": "string",
      "user.id": "string",
      "user_agent.browser": "string",
      "user_agent.browser_version": "string",
      "user_agent.platform": "string"
    },
    "url": "/api/projects/<PROJECT_ID>/collections/classes"
  }
]

读取单个集合信息

请求

GET /api/projects/PROJECT_ID/collections/COLLECTION_NAME?api_key=READ_KEY

返回

HTTP/1.1 200 OK
Content-Type: application/json; charset=utf-8

{
    "name": "trainers",
    "properties": {
      "item.id": "number",
      "item.on_sale": "boolean",
      "item.price": "number",
      "quantity": "number",
      "screen.name": "string",
      "user.id": "number",
      "user.level": "number",
      "user.referring_source": "string"
    },
    "url": "/api/projects/<PROJECT_ID>/collections/trainers"
}

记录多条数据

请求

POST /api/projects/PROJECT_ID/collections?api_key=WRITE_KEY
Content-Type: application/json; charset=utf-8

{
      "trainers": [
        { "name" : "何江" },
        { "name" : "John" }
      ],
      "classes": [
        { "price": 10 },
        { "price": 20 }
      ]
}

返回

HTTP/1.1 200 OK
Content-Type: application/json; charset=utf-8

{
  "trainers": [
    {
      "success": true
    },
    {
      "success": true
    }
  ],
  "classes": [
    {
      "success": true
    },
    {
      "success": true
    }
  ]
}

记录单条数据

请求

POST /api/projects/PROJECT_ID/collections/COLLECTION_NAME?api_key=WRITE_KEY
Content-Type: application/json; charset=utf-8

{
    "name" : "John"
}

返回

HTTP/1.1 200 OK
Content-Type: application/json; charset=utf-8

{
   "success": true
}

删除一个集合

请求

DELETE /api/projects/PROJECT_ID/collections/COLLECTION_NAME?api_key=MASTER_KEY

返回

HTTP/1.1 200 OK

删除一个属性

请求

DELETE /api/projects/PROJECT_ID/collections/COLLECTION_NAME/properties/PROPERTY_NAME?api_key=MASTER_KEY

返回

HTTP/1.1 200 OK

自动属性

自动属性会自动添加到每一条记录。

{
  "record": {
    "created_at": "2012-12-14T20:24:01.123000+00:00",
    "timestamp": "2012-12-14T20:24:01.123000+00:00",
    "id": "asd9fadifjaqw9asdfasdf939"
  },
  "device": {
    "id": "0980980231sdfsss",
    "model": "iPhone 6s plus"
  }
}

特别属性

当属性值为以下内容时，会被自动替换

Placeholder(string) Description
${client.ip} Replaced with the IP address of the client.
${client.user_agent} Replaced with the user agent string of the client.
